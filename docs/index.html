Developer Friendly

Using IoC? WSB integrates with all major .NET containers. Not using a container? No problem, WSB can run without one.

WSB was designed from the ground up to be frictionless in any .NET development environment.

Transport Independant

WSB supports RabbitMQ, and MSMQ as persistant transport brokers. A in-memory loopback transport is available for unit tests.

Extending transports is trivial thanks to a easy to implement transport interface.
