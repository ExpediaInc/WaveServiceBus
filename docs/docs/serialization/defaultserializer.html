Serilization is the process of taking a object and converting it into a representation that can be sent across the network. Wave uses serialization when moving a message from a publisher to its subscribers.

Out of the box, Wave uses the WCF DataContract Json serializer to achieve this. However, Wave also provides support for the most popular .NET serialization frameworks via additional NuGet packages.

Warning

All serializers work slightly differently, even if they use the same underlying format. Because of this, it is highly recommended that all publishers and subscribers use the same serializer.

The next sections will explain how to integrate with these 3rd party serializers.

If your infrastructure uses a custom serialization system, it is straight forward to create an adapter to bridge Wave serialization into your system.
