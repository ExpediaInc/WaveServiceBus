Json.net is a very popular JSON serialization library that is used by out of the box by ASP.NET WebAPI and in many other projects.

To use Json.net, add this NuGet package to your project:

PM> Install-Package Wave.Serialization.JsonDotNet

With this package installed, enabling Json.net as the serializer can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} var host = ServiceBus.ConfigureForHosting(c => { c.UseJsonDotNet(); }); {% endhighlight %}
