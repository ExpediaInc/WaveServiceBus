Out of the box, Wave only supports the WCF JSON DataContract Serializer. This package adds XML support using the WCF XML DataContract Serializer

To enable XML serialization add this NuGet package to your project:

PM> Install-Package Wave.Serialization.Xml

With this package installed, enabling XML Serialization can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} var host = ServiceBus.ConfigureForHosting(c => { c.UseXMLSerialization(); }); {% endhighlight %}
