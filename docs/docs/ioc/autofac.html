To use Autofac, add this NuGet package to your project:

PM> Install-Package Wave.IoC.AutoFac

With this package installed, enabling Autofac as the IoC container can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} // Create and configure the Autofac container var builder = new ContainerBuilder(); builder.RegisterType().As(); var container = builder.Build(); var host = ServiceBus.ConfigureForHosting(c => { // Pass the container to wave c.UseAutoFac(container); }); {% endhighlight %}
