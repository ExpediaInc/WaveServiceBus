To use StructureMap, add this NuGet package to your project:

PM> Install-Package Wave.IoC.StructureMap

With this package installed, enabling StructureMap as the IoC container can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} // Create and configure the StructureMap container var container = new Container(); container.Configure(x => x.For().Use()); var host = ServiceBus.ConfigureForHosting(c => { // Pass the container to wave c.UseStructureMap(container); }); {% endhighlight %}
