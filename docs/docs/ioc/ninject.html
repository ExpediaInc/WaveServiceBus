To use NInject, add this NuGet package to your project:

PM> Install-Package Wave.IoC.NInject

With this package installed, enabling NInject as the IoC container can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} // Create and configure the NInject kernel var kernel = new StandardKernel(); kernel.Bind().To(); var host = ServiceBus.ConfigureForHosting(c => { // Pass the NInject kernel to wave c.UseNInject(kernel); }); {% endhighlight %}
