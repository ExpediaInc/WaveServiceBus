To use Castle Windsor, add this NuGet package to your project:

PM> Install-Package Wave.IoC.CastleWindsor

With this package installed, enabling Castle Windsor as the IoC container can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} // Create and configure the Castle Windsor container var container = new WindsorContainer(); container.Register(Component.For().ImplementedBy()); var host = ServiceBus.ConfigureForHosting(c => { // Pass the container to wave c.UseCastleWindsor(container); }); {% endhighlight %}
