To use Unity, add this NuGet package to your project:

PM> Install-Package Wave.IoC.Unity

With this package installed, enabling Unity as the IoC container can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} // Create and configure the Unity container var container = new UnityContainer(); container.RegisterType(); var host = ServiceBus.ConfigureForHosting(c => { // Pass the container to wave c.UseUnity(container); }); {% endhighlight %}
