Wave can be hosted in a variety of service containers, depending on the use case, production environment, and general need.

Out of the box, Wave provides a simple console service host. This will run your Wave service as a console application. This is generally adequate for development, but lacks the robust management aspects of more complex service hosts.



If you not specify a service host, Wave will use the default one.

The following sections will go into detail around other service hosting options that are better suited for production use.
