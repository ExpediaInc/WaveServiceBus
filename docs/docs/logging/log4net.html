Log4Net is a popular logging framework that is used by many enterprises.

To integrate with Log4Net, add this NuGet package to your project:

PM> Install-Package Wave.Logging.Log4Net

With this package installed, enabling Log4Net support can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} var host = ServiceBus.ConfigureForHosting(c => { c.UseLog4Net(); }); {% endhighlight %}

Or specified using XML configuration in the host's configuration file:

{% endhighlight %}
