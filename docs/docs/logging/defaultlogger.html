Wave logs some diagnostics information on startup, and will log exceptions etc. By default, Wave will log to the console:



This is fine for development purposes, but is not flexible for production use. You may want to log to the system event log, or to a rolling file etc.

Rather than re-invent the wheel, Wave is designed to easily integrate with existing logging frameworks. It is likely that a preferred logger is already part of your infrastructure. The next sections demonstrate how to integrate with the most popular .NET logging frameworks.

If your infrastructure uses a custom logging system, it is straight forward to create an adapter to bridge Wave logging into your system.
