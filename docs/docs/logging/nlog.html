NLog is a popular logging framework that is used by many enterprises.

To integrate with NLog, add this NuGet package to your project:

PM> Install-Package Wave.Logging.NLog

With this package installed, enabling NLog support can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} var host = ServiceBus.ConfigureForHosting(c => { c.UseNLog(); }); {% endhighlight %}

Or specified using XML configuration in the host's configuration file:

{% endhighlight %}
