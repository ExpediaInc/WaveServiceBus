Commons Logging is a logging framework abstraction that is used by many enterprises.

To integrate with Commons Logging, add this NuGet package to your project:

PM> Install-Package Wave.Logging.CommonsLogging

With this package installed, enabling Commons Logging support can be done using fluent configuration when configuring the host:


{% highlight c# linenos %} var host = ServiceBus.ConfigureForHosting(c => { c.UseCommonsLogging(); }); {% endhighlight %}

Or specified using XML configuration in the host's configuration file:

{% endhighlight %}
